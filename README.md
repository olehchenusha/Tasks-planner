# Tasks planner

This app is tasks planner based on telegram bot.

## Quick Start

```shell
python3 -m venv .venv
source .venv/bin/activate
pip install --requirement requirements.txt
```

## .env

```shell
BOT_TOKEN= 

DB_NAME= 
DB_PASS= 
DB_USER= 
DB_HOST= 
```

## Launch

To start bot
```shell
python3 main.py
```
To start flask
```shell
python3 app.py
```
